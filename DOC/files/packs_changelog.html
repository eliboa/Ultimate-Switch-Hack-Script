Changelog des packs de l'Ultimate Switch Hack Script

 

V84

  • Mise à jour des scripts et configurations de EdiZon.

V83

  • Mise à jour de Atmosphere vers la dernière version du Github dans les packs Atmosphere et Kosmos, compilé par moi-même. Attention, cette version d'Atmosphere ne supporte pas encore le firmware 7.0.0.
  • Ajout du module Freebird (module permettant l'overclock de la Switch) pour tous les packs. Le homebrew de gestion de ce module, "Freeset", a également été ajouté.
  • Mise à jour de EdiZon en version 2.2.0.
  • Suppression du pack Kosmos spécifique au firmware 1.0.0 car le pack principal permettra maintenant de lancer ce pack également pour le firmware 1.0.0.
